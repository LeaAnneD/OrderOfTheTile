# Images Folder — The Order of the Tile

Place your image files in this folder and name them exactly as listed below.
The HTML files are already pointing to these exact filenames.

---

## Required Images

| Filename | Description | Used On |
|----------|-------------|---------|
| envelope.png | The sealed envelope logo — Cloud Dancer background (#F8F6F2), full oval, no transparency needed | Homepage hero |
| seal.png | The wax seal — transparent background PNG | Nav (all pages), Footer (all pages), Subscribe section, Story divider |

---

## Notes

- envelope.png must have a #F8F6F2 (Cloud Dancer) background baked in — not transparent
- seal.png must have a transparent background
- Both files should be PNG format
- Keep the original high resolution files — the browser will scale them correctly
- Once you place these files here, the images will display automatically when you open any HTML file in your browser

---

## Future Images

As you add photography or additional brand assets, add them here and reference them in the HTML.
Suggested naming convention:
- letter-hero-[issue-number].jpg — hero image for individual letter pages
- community-[name].jpg — headshots or brand photos for community page
